## 一、概述

统一工具层是 Living Tree AI Agent 的核心基础设施，提供统一的工具注册、发现、调用机制。

### 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                    智能体层 (Agent)                       │
│  - HermesAgent, EIAgent, SkillEvolutionAgent           │
└─────────────────────┬───────────────────────────────────┘
                      │ 调用
                      ↓
┌─────────────────────────────────────────────────────────┐
│              统一工具层 (ToolRegistry)                    │
│  - 工具注册/注销/查找/执行                              │
│  - 支持分类、标签、语义搜索                              │
└─────────────────────┬───────────────────────────────────┘
                      │ 路由
                      ↓
┌─────────────────────────────────────────────────────────┐
│                  工具实现层 (Tools)                      │
│  - 18+ 已实现工具                                       │
│  - P0/P1/P2 待实现工具                                  │
└─────────────────────────────────────────────────────────┘
```

## 二、快速开始

### 2.1 使用统一注册中心

```python
from client.src.business.unified_tool_registry import ToolRegistry

# 获取单例
registry = ToolRegistry.get_instance()

# 注册工具

# 执行工具

# 搜索工具

```

### 2.2 使用 @tool 装饰器

```python
from client.src.business.unified_tool_registry import tool, ToolRegistry

@tool(
    name="my_tool",
    description="我的工具",
    parameters={
        "type": "object",
        "properties": {
            "query": {"type": "string", "description": "查询内容"}
        },
        "required": ["query"]
    },
    toolset="custom"
)
def my_tool_handler(context, **args):
    return {"result": f"查询: {args['query']}"}
```

## 三、创建新工具

### 3.1 基础工具模板

```python
"""
MyTool - 我的工具

功能描述...
"""

from client.src.business.tools.base_tool import BaseTool
from client.src.business.tools.tool_result import ToolResult
from loguru import logger


class MyTool(BaseTool):
    """
    我的工具
    
    功能详细描述
    """
    
    def __init__(self):
        super().__init__(
            name="my_tool",                    # 工具名称（唯一）
            description="工具描述，用于 LLM 理解",  # 描述
            category="category_name",          # 分类
            tags=["tag1", "tag2"]              # 标签
        )
    
    def execute(self, **kwargs) -> ToolResult:
        """执行工具"""
        try:
            # 获取参数
            param1 = kwargs.get("param1")
            
            if not param1:
                return ToolResult.fail(error="param1 is required")
            
            # 执行逻辑
            result = do_something(param1)
            
            return ToolResult.ok(
                data={"result": result},
                message=f"Success: {result}"
            )
            
        except Exception as e:
            logger.error(f"MyTool execution failed: {e}")
            return ToolResult.fail(error=str(e))
    
    def health_check(self) -> bool:
        """健康检查"""
        return True
    
    def get_capabilities(self) -> dict:
        """获取工具能力描述"""
        return {
            "name": self.name,
            "category": self.category,
            "features": ["feature1", "feature2"]
        }


# ── 自动注册 ─────────────────────────────────────────────────────

def _auto_register():
    """自动注册到 ToolRegistry"""
    try:
        from client.src.business.tools.tool_registry import ToolRegistry
        registry = ToolRegistry.get_instance()
        
        tool = MyTool()
        if registry.register_tool(tool):
            logger.info(f"Auto-registered: {tool.name}")
            return True
    except Exception as e:
        logger.error(f"Auto-registration error: {e}")
    return False
```

## 五、工具注册系统

### 5.1 自动注册

所有工具都应有 `_auto_register()` 函数，在模块加载时自动注册：

```python
# 工具模块加载时自动注册
_auto_register()
```

### 5.2 手动注册

```python
from client.src.business.unified_tool_registry import ToolRegistry

registry = ToolRegistry.get_instance()

# 注册 BaseTool 实例
registry.register_tool(my_tool_instance)

# 注册函数
registry.register_tool(
    name="my_func",
    description="描述",
    handler=my_function
)
```

### 5.3 查看已注册工具

```python
registry = ToolRegistry.get_instance()

# 列出所有工具
for tool in registry.list_tools():
    print(f"[{tool.category}] {tool.name}")

# 搜索工具
results = registry.search_tools("天气")

# 获取统计
stats = registry.stats()
print(f"Total: {stats['total_tools']}")
```

## 九、最佳实践

### 9.1 命名规范

- 工具名称：小写 + 下划线（如 `my_tool`）
- 类名：PascalCase（如 `MyTool`）
- 描述：清晰、详细，包含同义词

### 9.2 错误处理

```python
def execute(self, **kwargs) -> ToolResult:
    try:
        # 执行逻辑
        pass
    except ValueError as e:
        return ToolResult.fail(error=f"参数错误: {e}")
    except Exception as e:
        logger.exception("Unexpected error")
        return ToolResult.fail(error=str(e))
```

### 9.3 健康检查

```python
def health_check(self) -> bool:
    """检查工具是否可用"""
    try:
        # 检查依赖
        check_dependency()
        return True
    except:
        return False
```
