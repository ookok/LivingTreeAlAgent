# Skill: P2P模型下载引导

> 当用户需要下载模型时触发此引导

## 触发条件
- 用户点击"下载模型"
- 用户说"安装XXX模型"
- 系统推荐模型优化

## 引导流程

### 步骤1：理解需求
```
询问用户：
- 需要什么类型的模型？（气象、环保、通用）
- 用途是什么？（本地推理、参考、分享）
- 存储空间限制？
```

### 步骤2：搜索可用模型
```python
# 搜索模型
p2p_search_models(query="用户需求描述", category="可选")
```

### 步骤3：展示选项
显示模型信息：
- 模型名称和ID
- 大小和系统要求
- 来源（本地邻居/中继服务器）
- 健康度/可用性

### 步骤4：确认并下载
```python
# 开始下载
p2p_download_model(model_id="选择的模型ID", priority="high/normal")
```

### 步骤5：跟踪进度
```python
# 轮询进度
p2p_get_download_progress(task_id="任务ID")
```

### 步骤6：完成通知
- 下载完成
- 自动验证模型完整性
- 更新模型列表

## P2P分发优势
相比传统下载：
- ✓ 从邻居节点获取，速度更快
- ✓ 支持断点续传
- ✓ 分布式存储，节省带宽

## 常见问题

### Q: 下载速度慢
A: 可能是邻居节点少，等待Gossip扩散或手动连接更多节点

### Q: 下载中断
A: 系统自动记录进度，重新调用下载时继续

### Q: 模型验证失败
A: 重新下载或从其他节点获取

## 成功标准
- 模型下载完成
- 模型通过SHA256校验
- 模型可正常加载运行
