# Pngjpeg

> Generated by Hermes CLI-Anything

## Overview

批量转换图片格式工具，支持PNG和JPEG互转

## Features

- **run**: 执行操作

## Quick Start

### Installation

```bash
# Install from source
pip install -e .

# Or build distribution package
python -m build
pip install dist/*.tar.gz
```

### Basic Usage

```bash
# Show help
pngjpeg --help

# Verbose mode
pngjpeg -v run --input ./data

# Specify output directory
pngjpeg -o ./output run --input ./data
```

## Command Reference

### run / convert

Convert file formats

```bash
pngjpeg run --input ./input.txt --format pdf
```

Arguments:
- `--input, -i`: Input file or directory (required)
- `--format, -f`: Target format
- `--recursive, -r`: Process subdirectories recursively

### batch

Batch process multiple files

```bash
pngjpeg batch --input ./files --recursive
```

### download

Download remote resources

```bash
pngjpeg download --input https://example.com/file.zip
```

## Configuration

Environment variables:

- `CLI_TOOL_VERBOSE`: Enable verbose output
- `CLI_TOOL_OUTPUT`: Default output directory
