# 智能搜索联想模块 - SearchSuggestion

## 概述

类似百度搜索建议的智能联想功能，在用户输入时实时显示搜索建议。

## 功能特性

1. **实时联想**：输入时自动触发，300ms 防抖
2. **10行显示**：纵向排列，最多显示10条建议
3. **时间排序**：按搜索热度/时间排序，最新的排在最上面
4. **知识库驱动**：调用本地知识库获取建议
5. **本地历史**：结合搜索历史提供个性化建议
6. **键盘导航**：支持上下键选择，Enter确认

## 架构

```
search_suggestion/
├── __init__.py           # 模块入口
├── suggestion_popup.py   # 联想弹窗组件
├── suggestion_model.py   # 建议数据模型
├── knowledge_query.py    # 知识库查询接口
└── cache.py              # 建议缓存
```

## 数据源

| 数据源 | 优先级 | 说明 |
|--------|--------|------|
| 本地缓存 | 1 | 最近搜索历史（最新优先） |
| 知识库 | 2 | Chroma/FAISS 语义搜索 |
| 网络API | 3 | 外部搜索建议API |

## 交互流程

```
用户输入 → 300ms防抖 → 查询缓存 → 查询知识库 → 合并排序 → 显示联想
     ↓
用户选择 → 填充输入框 → 执行搜索
```

## 技术要点

1. **防抖处理**：300ms 延迟，避免频繁请求
2. **异步查询**：非阻塞式知识库查询
3. **缓存策略**：LRU 缓存 + TTL 过期
4. **模糊匹配**：支持前缀匹配和拼音匹配
5. **热度衰减**：历史搜索按时间衰减权重
