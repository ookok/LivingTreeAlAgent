```
┌─────────────────────────────────────────────────────────────────┐
│                    Evolution-Intent Bridge                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  IntentEngine ────────────────────────► EvolutionEngine         │
│  ┌─────────────┐                        ┌─────────────────────┐ │

│  │ 建议模型    │◄──模型推荐───────────────│ suggested_model     │ │
│  └─────────────┘                        └─────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

### 核心类

| 类 | 功能 |
|-----|------|
| `EvolutionIntentBridge` | 桥接器主类 |
| `IntentSignal` | 意图信号（Intent → Evolution） |
| `ExecutionFeedback` | 执行反馈（闭环学习） |
| `LearningEnhancement` | 学习增强（Evolution → Intent） |

### API 使用

```python
from core.evolution_engine.bridge import create_bridge, create_full_bridge

# 方式1: 创建空桥接器
bridge = create_bridge()

# 方式2: 创建完整桥接器（自动初始化两个引擎）
bridge = create_full_bridge()

# 处理意图
intent = bridge.process_intent("帮我写一个用户登录接口")

# 报告执行结果
from core.evolution_engine.bridge import ExecutionFeedback
bridge.report_execution_result(ExecutionFeedback(
    intent_id="test-001",
    success=True,
    duration_ms=1500,
    output="Done",
    quality_score=0.92,
))

# 获取桥接器状态
status = bridge.get_bridge_status()
```

### 联动场景

| 场景 | 方向 | 说明 |
|------|------|------|
| 意图解析 | Intent → Evolution | 用户意图触发 Evolution 信号 |
| 执行反馈 | Intent → Evolution | 执行结果反馈给 LearningEngine |
| 学习增强 | Evolution → Intent | 从历史学习提供洞察和警告 |
| 风险提示 | Evolution → Intent | 低置信度意图的风险警告 |
| 模型推荐 | Evolution → Intent | 基于历史的模型推荐 |
