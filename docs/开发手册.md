## 1. 开发环境搭建

### 1.1 系统要求

- Windows 10+ / macOS 10.14+ / Ubuntu 18.04+
- Python 3.11+
- Git
- 至少 8GB RAM (推荐 16GB+)
- PyQt6 支持的显卡 (可选，用于 GPU 加速)

### 1.2 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/your-repo/hermes-desktop.git
cd hermes-desktop

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 安装 PyQt6
pip install PyQt6

# 5. 安装中继服务器依赖
cd server/relay_server
pip install -r requirements-server.txt
```

### 1.3 启动方式

```bash
# 启动桌面客户端 (默认)
python main.py client

# 启动中继服务器
python main.py relay

# 启动追踪服务器
python main.py tracker

# 启动所有服务
python main.py all
```

## 3. 核心模块开发

### 3.1 创建新模块

1. 在 `core/` 下创建模块目录
2. 创建 `__init__.py` 导出核心类和函数
3. 在 `ui/` 下创建对应的 PyQt6 面板
4. 在 `main_window.py` 中集成新 Tab

**示例: 创建 `core/example_module/`**

```python
# core/example_module/__init__.py
from .example import ExampleClass

__all__ = ["ExampleClass"]
```

```python
# core/example_module/example.py
class ExampleClass:
    def __init__(self):
        self.name = "Example Module"

    def process(self, data):
        return f"Processed: {data}"
```

### 3.2 模块设计原则

1. **单例模式**: 使用 `get_xxx()` 函数返回单例
2. **配置驱动**: 使用 `config/` 下的配置文件
3. **数据库迁移**: 在 `database/migrations.py` 添加新版本
4. **信号通信**: 使用 PyQt6 信号进行组件间通信
5. **异步优先**: IO密集型操作使用 asyncio

### 3.3 模块集成流程

```python
# 1. 导入模块
from core.example_module import ExampleClass

# 2. 获取单例
example = ExampleClass()

# 3. 调用方法
result = example.process("test data")

# 4. 在 UI 中集成
# ui/example_panel.py
from PyQt6.QtWidgets import QWidget, QVBoxLayout, QLabel
from core.example_module import ExampleClass

class ExamplePanel(QWidget):
    def __init__(self):
        super().__init__()
        self.example = ExampleClass()
        self.setup_ui()

    def setup_ui(self):
        layout = QVBoxLayout()
        layout.addWidget(QLabel("Example Panel"))
        self.setLayout(layout)
```

## 4. UI 面板开发

### 4.1 PyQt6 基础

```python
from PyQt6.QtWidgets import (
    QApplication, QMainWindow, QWidget,
    QVBoxLayout, QHBoxLayout, QPushButton,
    QLabel, QTextEdit, QTabWidget
)
from PyQt6.QtCore import pyqtSignal, QThread

class ExampleWorker(QThread):
    finished = pyqtSignal(object)

    def run(self):
        # 后台任务
        result = self.process()
        self.finished.emit(result)
```

### 4.2 面板模板

```python
# ui/example/example_panel.py
from PyQt6.QtWidgets import (
    QWidget, QVBoxLayout, QHBoxLayout,
    QPushButton, QLabel, QTextEdit
)
from PyQt6.QtCore import pyqtSignal

class ExamplePanel(QWidget):
    """示例面板"""

    # 信号定义
    data_changed = pyqtSignal(dict)

    def __init__(self, parent=None):
        super().__init__(parent)
        self.init_ui()

    def init_ui(self):
        """初始化UI"""
        layout = QVBoxLayout()

        # 标题
        title = QLabel("示例面板")
        title.setStyleSheet("font-size: 16px; font-weight: bold;")
        layout.addWidget(title)

        # 内容区域
        self.content = QTextEdit()
        layout.addWidget(self.content)

        # 按钮区域
        btn_layout = QHBoxLayout()
        self.process_btn = QPushButton("处理")
        self.process_btn.clicked.connect(self.on_process)
        btn_layout.addWidget(self.process_btn)

        layout.addLayout(btn_layout)
        self.setLayout(layout)

    def on_process(self):
        """处理按钮点击"""
        text = self.content.toPlainText()
        # 处理逻辑
        self.data_changed.emit({"result": text})
```

### 4.3 集成到主窗口

from fastapi import APIRouter, HTTPException

from typing import Dict, Any

router = APIRouter(prefix="/api/example", tags=["example"])

class ExampleRequest(BaseModel):

async def process_example(request: ExampleRequest):

        "name": request.name,
        "result": f"Processed {len(request.data)} items"
    }

## 6. 数据库

### 6.1 数据库迁移

```python
# database/migrations.py
MIGRATIONS = {
    "v1": """
        CREATE TABLE IF NOT EXISTS sessions (
            id INTEGER PRIMARY KEY,
            created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
        );
    """,
    "v2": """
        CREATE TABLE IF NOT EXISTS messages (
            id INTEGER PRIMARY KEY,
            session_id INTEGER,
            content TEXT,
            FOREIGN KEY (session_id) REFERENCES sessions(id)
        );
    """,
    # 添加新的迁移...
}

def run_migration(version: str, conn):
    """执行迁移"""
    if version in MIGRATIONS:
        cursor = conn.cursor()
        cursor.execute(MIGRATIONS[version])
        conn.commit()
```

### 6.2 使用 SQLite

```python
import sqlite3
from pathlib import Path

DB_PATH = Path.home() / ".hermes-desktop" / "hermes.db"

def get_connection():
    """获取数据库连接"""
    DB_PATH.parent.mkdir(parents=True, exist_ok=True)
    conn = sqlite3.connect(DB_PATH)
    conn.row_factory = sqlite3.Row
    return conn

def query_example():
    """查询示例"""
    conn = get_connection()
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM sessions LIMIT 10")
    rows = cursor.fetchall()
    conn.close()
    return [dict(row) for row in rows]
```

## 8. 编码规范

### 8.1 Python 规范

- 遵循 PEP 8
- 使用类型提示
- 文档字符串使用 Google 风格
- 导入排序: 标准库 > 第三方库 > 本地模块

**示例:**

```python
from typing import List, Optional, Dict
from pathlib import Path

import pandas as pd
from PyQt6.QtWidgets import QWidget

from core.example_module import ExampleClass


class ExampleWidget(QWidget):
    """示例组件。

    用于演示编码规范的示例组件。

    Attributes:
        name: 组件名称
        _count: 内部计数器
    """

    def __init__(self, name: str, parent: Optional[QWidget] = None) -> None:
        """初始化示例组件。

        Args:
            name: 组件名称
            parent: 父组件
        """
        super().__init__(parent)
        self.name = name
        self._count = 0

    def process(self, data: List[Dict]) -> Dict[str, int]:
        """处理数据。

        Args:
            data: 输入数据列表

        Returns:
            处理结果字典

        Raises:
            ValueError: 当数据为空时
        """
        if not data:
            raise ValueError("数据不能为空")

        self._count += len(data)
        return {
            "total": self._count,
            "items": len(data)
        }
```

### 8.2 Git 规范

**提交信息格式:**

```
<type>(<scope>): <subject>

<body>

<footer>
```

**类型:**
- feat: 新功能
- fix: 修复bug
- docs: 文档
- style: 格式
- refactor: 重构
- test: 测试
- chore: 杂项

**示例:**

```
feat(credit_economy): 添加积分充值功能

- 实现用户积分充值接口
- 添加VIP等级计算逻辑
- 集成支付网关回调

Closes #123
```

### 8.3 文件命名

- Python 文件: `snake_case.py`
- PyQt6 文件: `xxx_panel.py`, `xxx_widget.py`
- 测试文件: `test_xxx.py`
- 配置文件: `xxx_config.py`

### B. 调试技巧

2. **FastAPI 调试**: 使用 `--reload` 参数

### C. 常见问题

1. **PyQt6 导入失败**: 确保已安装 PyQt6
2. **模块找不到**: 检查 `sys.path` 是否包含项目根目录
3. **数据库锁定**: 确保没有其他进程占用数据库
