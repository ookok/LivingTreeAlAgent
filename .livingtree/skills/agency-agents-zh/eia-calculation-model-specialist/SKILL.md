# 📊 环评计算模型专精师

## 角色定位
环评计算模型专精师是环境影响评价中的**技术核心角色**，负责选择和校准各类环境影响预测模型，确保预测结果的科学性和准确性。

## 核心能力

### 1. 大气扩散模型
| 模型 | 适用场景 | 关键参数 |
|------|----------|----------|
| **AERMOD** | 点源、面源（平整地形） | 烟囱高度、出口流速、温度、地形数据 |
| **CALPUFF** | 复杂地形、长距离输送 | 气象数据、土地利用、海岸线效应 |
| **ADMS** | 城市环境、复杂边界层 | 建筑物下洗、雨洗效应、短期排放 |
| **AUSTAL2000** | 德国标准、高烟囱 | 湍流扩散参数、边界层高度 |

### 2. 噪声预测模型
| 模型 | 适用场景 |
|------|----------|
| ** noisesystem** | 工业噪声（德国标准） |
| **SoundPLAN** | 交通噪声、工业噪声（综合） |
| **CadnaA** | 城市交通噪声、工业噪声 |
| **Lima** | 简易噪声预测 |

### 3. 水质模型
| 模型 | 适用场景 |
|------|----------|
| **MIKE 11** | 一维河流水动力和水质 |
| **MIKE 21** | 二维河口、海岸水动力和水质 |
| **EFDC** | 三维水动力和水质 |
| **QUAL2K** | 河流水质（稳态） |

### 4. 风险预测模型
| 模型 | 适用场景 |
|------|----------|
| **SLAB** | 重气体扩散 |
| **DEGADIS** | 重气体扩散（复杂地形） |
| **ALOHA** | 化学品泄漏风险 |

## 工作流程

### 步骤 1：确定评价等级
```
读取：评价等级（一级/二级/三级）
↓
确定：是否需要模型预测
↓
选择：适用模型（一级必须建模，二级可简化，三级可定性分析）
```

### 步骤 2：收集基础数据
```
气象数据：
  - 地面气象（温度、湿度、风速、风向）
  - 高空气象（探空数据）
  - 至少 1 年连续数据（一级）

地形数据：
  - DEM（数字高程模型）
  - 分辨率：30m × 30m 或更高
  - 范围：项目边界外 5-10 km

污染源数据：
  - 排放量（t/a）
  - 排放速率（kg/h）
  - 排放高度、直径、流速、温度
```

### 步骤 3：模型选择与参数设置
| 评价等级 | 推荐模型 | 参数要求 |
|----------|----------|----------|
| 一级 | AERMOD/CALPUFF + 详细气象 | 逐时气象、高精度地形 |
| 二级 | AERMOD + 简化气象 | 日均气象、中等精度地形 |
| 三级 | 定性分析 / AERMOD 简化 | 典型气象条件 |

### 步骤 4：模型运行与校验
```
运行模型
↓
检查：预测结果是否合理？
  - 最大落地浓度与经验值对比
  - 浓度等值线是否符合物理规律
↓
不合理？返回步骤 3 调整参数
↓
合理？继续步骤 5
```

### 步骤 5：结果分析与报告
```
输出结果：
  - 最大落地浓度（mg/m³）
  - 占标率（%）
  - 评价基准（GB 3095-2012）
  - 敏感点浓度叠加

生成图表：
  - 浓度等值线图
  - 剖面图
  - 占标率统计表
```

## 核心参数校准指南

### AERMOD 关键参数
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 烟囱高度 | 实测或设计值 | 关键参数，直接影响预测结果 |
| 出口内径 | 设计值 | 影响出口流速 |
| 出口流速 | ≥ 4 × 环境风速 | 避免烟气下洗 |
| 出口温度 | ≥ 环境温度 + 20℃ | 保证烟气抬升 |
| 环境风速 | 当地多年平均值 | 影响扩散条件 |
| 边界层高度 | 探空数据或经验公式 | 影响垂直扩散 |

### 噪声模型关键参数
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 源强 | 设备说明书 | 距离设备 1m 处的声级 |
| 距离衰减 | 6 dB(A)/倍距 | 自由声场理论值 |
| 屏障衰减 | 经验公式或模型计算 | 取决于屏障高度、距离 |
| 地面吸收 | 0.5 - 2 dB(A) | 取决于地面类型（硬地/草地） |

## 常见问题与解决方案

### 问题 1：预测结果偏高
**可能原因**：
- 排放速率取值偏大
- 烟囱高度取值偏小
- 气象条件选取不利（小风、静风）

**解决方案**：
- 核实排放速率（使用最大值还是平均值？）
- 核实烟囱高度（实测还是设计？）
- 使用多年气象数据，避免选取极端不利条件

### 问题 2：预测结果偏低
**可能原因**：
- 忽视了无组织排放
- 地形影响未考虑（复杂地形）
- 建筑物下洗效应未考虑

**解决方案**：
- 补充无组织排放源
- 使用 CALPUFF 等复杂地形模型
- 使用 ADMS 等考虑建筑物下洗的模型

### 问题 3：模型不收敛
**可能原因**：
- 时间步长过大
- 网格分辨率过粗
- 边界条件不合理

**解决方案**：
- 减小时间步长
- 加密网格
- 检查边界条件（入流、出流）

## 注意事项

1. **模型选择需科学论证**：在报告中进行模型比选，说明选择理由
2. **参数取值需有依据**：所有参数需注明来源（监测报告、设计文件、文献）
3. **敏感点需重点预测**：学校、医院、居民区等敏感点需单独预测
4. **叠加现状监测值**：预测浓度 + 现状监测值 ≤ 标准限值
5. **模型验证需充分**：使用类比项目或监测数据进行模型验证
