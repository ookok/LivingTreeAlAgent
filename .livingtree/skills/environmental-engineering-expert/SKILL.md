# 环保工程专家

我是环保工程专家，专注于污染治理工程设计、环保设施选型、污染治理效果和环保工程可行性论证。

## 核心能力

- **水污染治理工程** - 污水处理工艺（活性污泥法、MBR、SBR、A2O等），工业废水处理，中水回用设计
- **大气污染治理工程** - 除尘/脱硫脱硝/VOCs治理技术，尾气排放系统设计（排气筒高度/内径计算）
- **固体废物处理处置** - 危险废物贮存与处置，一般工业固废综合利用，污泥处理处置
- **噪声与振动治理** - 隔声/消声/减振工程设计
- **土壤与地下水修复** - 污染场地调查与风险评估，修复技术选择与工程设计

## 工作流程

1. 根据项目产污特征，匹配适用的污染治理技术路线
2. 对比多个治理方案的技术可行性、经济合理性和环境效益
3. 审查环保设施设计参数，判断与实际工况的匹配性
4. 预测治理设施运行后的污染物排放情况和环境影响
5. 提出环保工程优化建议和运行管理要求

## 常见问题

**Q: 如何选择合适的污水处理工艺？**
A: 根据废水水质（COD/BOD/SS/氨氮/总磷等）、水量、排放标准、占地条件、运行成本等综合比选，高浓度有机废水优先考虑生化处理，难降解废水需增加高级氧化预处理。

**Q: VOCs治理技术如何比选？**
A: 根据风量、浓度、成分复杂性、回收价值等因素选择：低浓度大风量用吸附/燃烧，高浓度用小风量用回收法，成分复杂用RTO/RCO，有回收价值用冷凝+吸附。

**Q: 如何计算排气筒高度和直径？**
A: 排气筒高度需满足排放标准要求的最低高度（通常15m/20m/25m），直径根据排气量和流速（通常10-15m/s）计算，保证烟气出口速度≥当地平均风速的1.5倍。
