🏛️ 系统架构

✨ 核心功能

🧠 本地大模型

支持 Ollama、GPT4All、ModelScope 等本地模型，无需联网，数据完全私密。支持 Qwen、LLM-Research 等 GGUF 模型。

🌱 装配园 Assembly Garden

自动化知识与技能生成管线。从代码库、文档、博客中提取知识，自动生成可执行的 Skill，并沉淀到知识林地。

🌲 知识林地 Knowledge Grove

行业化知识管理，按电子/硬件/软件等12个行业分类。支持伪域名路由、Wiki渲染、LTKG包导入导出。

📦 技能市场 Skill Market

基于 SKILL.md Manifest 的技能分发系统。技能即插即用，支持版本管理、依赖解析、自动安装。

💰 积分经济系统

动态积分发放与智能消耗构建可持续 P2P 经济体系。VIP 等级、每日赠送、状态通道、预测市场。

🏆 全链路成就系统

每个行为都有成就，每次成长都有记录。从萌新到传奇的全过程可追溯、可展示、可传承。

🎮 挂机等级系统

让用户的每一秒在线时间都产生价值。10级体系、挂机小游戏、等级特权、排行榜。

🏢 通用资产交易

从数字到物理的完整社会模式。5种交付方式自动匹配、物流集成、信任网络、微型社会 DAO。

🔌 API 接口

GET /api/health

健康检查，返回服务状态

GET /api/stats/overview

全局统计概览

GET /api/stats/trends

进化趋势，按周统计

GET /api/config/sync

从服务器拉取配置

POST /api/config/sync

推送本地配置到服务器

GET /api/relay/params

获取中继参数
