💡 设计理念

离线优先 — 核心功能不依赖服务器，关闭服务器不影响本地使用。
轻量中继 — Relay Server 只做数据聚合和参数分发。

🏗️ 技术栈

FastAPI + Uvicorn + Pydantic + SQLite
纯静态 HTML 前端，无 PyQt 依赖

📂 数据目录

~/.hermes-desktop/relay_server/
包含 raw/、agg/、web/ 等子目录
