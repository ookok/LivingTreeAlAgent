## 技术要点

1. **懒加载模式**：`EmbeddingEngine` 改为首次使用时才加载模型，加快启动速度
2. **增量更新**：`ConfigHotReloader` 只更新变更配置项，避免全量清除缓存
3. **参数自适应**：遗传算法的 mutation_rate/crossover_rate 根据进化状态自动调整
4. **多阶段 Docker 构建**：镜像从 ~1.2GB 优化到 ~800MB
5. **Kubernetes 完整配置**：Deployment + HPA + Ingress + PVC 全套
