# 专家训练系统使用指南

## 概述

专家训练系统是基于训练内容自主创建专家角色的功能模块。系统可以根据提供的训练内容（文本、文档、对话记录等）自动提取专家特征，生成专家角色文件（SKILL.md），并按照行业分类进行组织管理。

## 核心功能

### 1. 单个专家训练

**功能描述**：根据输入的训练内容，自动创建或更新专家角色。

**使用方法**：

#### 方法一：通过UI界面

1. 打开"专家训练与管理面板"
2. 选择"单个训练"标签页
3. 在"训练内容输入"框中输入训练内容
   - 可以是对话记录、专业文档、知识库内容等
   - 内容越长，提取的特征越准确
4. （可选）在"专家名称"框中输入指定的专家名称
   - 如果不输入，系统会自动生成专家名称
5. 点击"开始训练"按钮
6. 等待训练完成，查看训练结果

**示例训练内容**：

```
我是环境影响评价专家，专注于建设项目环境影响评价报告的编制和审查。
熟悉《环境影响评价法》、《建设项目环境保护管理条例》等法律法规。
擅长大气环境影响预测、水环境影响分析、噪声影响评价等技术工作。
能够使用AERMOD、ADMS等大气扩散模型进行模拟预测。
```

#### 方法二：通过代码调用

```python
from client.src.business.expert_training.tools import train_expert_from_text

# 训练内容
content = """
我是数据分析专家，擅长Python、R、SQL等工具，能够进行统计分析、机器学习建模。
熟悉pandas、numpy、scikit-learn等库，能够处理结构化数据和非结构化数据。
"""

# 训练专家
result = train_expert_from_text(content, "数据分析专家")

print(result)
# 输出：
# {
#     "success": True,
#     "expert_name": "数据分析专家",
#     "expert_dir": ".../.livingtree/skills/agency-agents-zh/data-analysis-expert",
#     "skill_file": ".../.livingtree/skills/agency-agents-zh/data-analysis-expert/SKILL.md",
#     "is_update": False,
#     "industry_classification": {...},
#     "notification": {...}
# }
```

### 2. 批量训练

**功能描述**：从指定目录中批量读取训练文件，自动创建专家角色。

**支持的文件类型**：`.txt`, `.md`, `.pdf`, `.docx`, `.csv`, `.json`

**使用方法**：

#### 方法一：通过UI界面

1. 打开"专家训练与管理面板"
2. 选择"批量训练"标签页
3. 点击"浏览..."按钮，选择包含训练文件的目录
4. 点击"开始批量训练"按钮
5. 等待批量训练完成，查看训练结果

#### 方法二：通过代码调用

```python

#     "success": 8,
#     "failed": 1,
#     "skipped": 1,
```

### 3. 行业分类

**功能描述**：查看基于 GB/T 4754-2017 的国民经济行业分类体系。

**行业分类标准**：

系统采用《国民经济行业分类》国家标准（GB/T 4754-2017），包含：

- **20个门类**（如：A 农、林、牧、渔业，B 采矿业，C 制造业等）
- **97个大类**（如：13 农副食品加工业，14 食品制造业等）
- **473个中类**
- **1380个小类**

**查看行业分类**：

1. 打开"专家训练与管理面板"
2. 选择"行业分类"标签页
3. 查看行业分类树
4. 点击"刷新行业分类"按钮可以重新加载

**通过代码获取行业分类**：


### 5. 通知历史

**功能描述**：查看专家角色创建、更新、删除的通知历史。

**通知类型**：

- `expert_deleted`：专家删除通知

**查看通知历史**：

1. 打开"专家训练与管理面板"

4. 点击"刷新通知"按钮可以重新加载

**通过代码获取通知历史**：

```python

for notification in notifications:
    print(notification)
```

## 行业分类自动更新

### 功能描述

系统可以定期从政府公开数据（国家统计局网站）检查并更新行业分类标准。

### 更新间隔

默认24小时（86400秒）检查一次更新。

### 手动检查更新

```python
from client.src.business.expert_training.tools import check_industry_update

# 检查行业分类更新
result = check_industry_update()

print(result)
# 输出：
# {
#     "need_update": True,
#     "latest_version": "GB/T 4754-202X",  # 如果有新版本
#     "current_version": "GB/T 4754-2017",
#     "details": {...},
#     "need_confirmation": True
# }
```

### 自动更新配置

```python
from client.src.business.expert_training.industry_updater import get_industry_updater

# 获取更新器，设置检查间隔为1小时（3600秒）
updater = get_industry_updater(check_interval=3600)

# 检查并更新
result = updater.check_and_update(force=False)
```

## 通知系统

### 功能描述

当专家角色被创建、更新或删除时，系统会自动通知其他智能体。

### 通知方式

1. **文件通知**（默认）：写入通知文件，其他智能体轮询读取
2. **消息总线**（可选）：通过消息队列发布通知
3. **WebSocket推送**（可选）：实时推送给在线智能体

### 通知文件存储位置

通知文件存储在 `.livingtree/notifications/` 目录下，文件名为 `通知类型_时间戳.json`。

### 监听通知

```python
from client.src.business.expert_training.notification_system import get_notification_system

# 获取通知系统
notification_system = get_notification_system()

# 定义监听器函数
def my_listener(notification: dict):
    print(f"收到通知：{notification.get('message')}")

# 注册监听器
notification_system.register_listener(my_listener)

# 取消注册监听器
notification_system.unregister_listener(my_listener)
```

## 常见问题

### Q1: 训练内容需要多少字？

A: 建议至少200字以上，内容越详细，提取的专家特征越准确。

### Q2: 如何指定专家的行业分类？

A: 系统会根据训练内容自动匹配行业分类。如果需要手动指定，可以修改生成的 SKILL.md 文件中的 `industry_code` 和 `industry_name` 字段。

### Q3: 批量训练支持哪些文件类型？

A: 目前支持 `.txt`, `.md`, `.pdf`, `.docx`, `.csv`, `.json`。PDF和DOCX支持需要额外安装依赖库。

### Q4: 如何查看已训练的专家列表？

A: 可以通过以下代码获取：

```python
from pathlib import Path

# 专家目录
experts_dir = Path(".livingtree/skills/agency-agents-zh")

# 列出所有专家
for expert_dir in experts_dir.iterdir():
    if expert_dir.is_dir():
        skill_file = expert_dir / "SKILL.md"
        if skill_file.exists():
            print(f"专家：{expert_dir.name}")
```

### Q5: 通知系统会占用很多磁盘空间吗？

A: 系统会自动清理旧的通知文件，只保留最近100个通知文件。
