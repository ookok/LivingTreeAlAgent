| `docs/CONFIG_EXTERNALIZATION_PLAN.md` | 配置外部化方案文档 |

### 核心功能

1. **统一配置管理** (`UnifiedConfig`)
   - 单例模式线程安全
   - YAML配置文件加载
   - 环境变量自动替换 `${VAR_NAME}`
   - 分层配置合并 (defaults < unified.yaml < 环境变量)

2. **便捷API**
   ```python
   from core.config.unified_config import get_config, get
   
   config = get_config()
   config.get("endpoints.ollama.url")  # 获取配置
   config.get_ollama_url()              # Ollama地址快捷方法
   config.get_timeout("browser")        # 获取超时配置
   config.get_delay("polling_medium")   # 获取延迟配置
   ```

3. **配置结构**
   - `endpoints`: 服务端点 (ollama/cloud_sync/tracker/relay/webrtc)
   - `timeouts`: 超时配置 (default/long/browser/download/quick)
   - `retries`: 重试配置
   - `delays`: 延迟配置 (轮询间隔/周期任务/等待)
   - `batch`: 批处理配置
   - `limits`: 资源限制
   - `api_keys`: API Keys (支持环境变量)
   - `paths`: 路径配置
   - `llm`: LLM默认参数
