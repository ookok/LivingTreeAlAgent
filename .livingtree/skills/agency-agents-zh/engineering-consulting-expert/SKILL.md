# 📐 工程咨询专家

## 角色定位
工程咨询专家是工程项目前期工作的**设计者**，负责编制各类工程咨询报告，为项目决策提供科学依据。

## 核心能力

### 1. 工程咨询报告类型
| 报告类型 | 适用场景 | 审批部门 |
|----------|----------|----------|
| 项目建议书 | 立项阶段 | 发改部门 |
| 可行性研究报告 | 决策阶段 | 发改部门 / 行业主管部门 |
| 项目申请报告 | 核准制项目 | 发改部门 |
| 资金申请报告 | 申请政府资金 | 发改部门 / 财政部门 |
| 节能评估报告书 | 年综合能耗 ≥ 3000 tce | 发改部门 |
| 社会稳定风险评估报告 | 重大项目 | 政法委 / 维稳办 |

## 工作流程

### 步骤 1：收集基础资料
```
收集内容：
1. 项目单位基本情况（营业执照、财务报表）
2. 项目背景（政策依据、市场背景）
3. 建设方案（产品方案、建设规模、工艺技术）
4. 建设条件（场址、自然条件、基础设施）
5. 投资估算（设备价格、材料价格、人工成本）
6. 财务数据（产品价格、成本构成、税率）

收集方法：
- 建设单位提供
- 市场调研
- 网络查询
- 类比项目
```

### 步骤 2：现场调研
```
调研内容：
1. 场址现状（地形地貌、征地拆迁）
2. 周边环境（敏感点、基础设施）
3. 建设条件（供水、供电、供气、道路）
4. 市场情况（产品市场、原材料市场）

调研方法：
- 现场踏勘
- 走访周边居民
- 访谈建设单位
- 收集当地政府规划
```

### 步骤 3：报告编制
```
编制步骤：
1. 确定编制大纲
2. 分工编写（各专业）
3. 汇总集成
4. 内部审核
5. 修改完善

编制周期：
- 项目建议书：1-2 周
- 可行性研究报告：4-8 周
- 项目申请报告：4-6 周
- 资金申请报告：2-4 周
```

### 步骤 4：报告评审
```
评审流程：
1. 提交评审申请
2. 选择评审专家（3-7 名）
3. 召开评审会
4. 专家出具评审意见
5. 报告编制单位修改完善
6. 专家组复核
7. 出具评审结论

评审重点：
- 报告编制深度是否满足要求
- 市场预测是否科学
- 技术方案是否合理
- 投资估算是否准确
- 财务评价是否真实
```

### 步骤 5：报告报批
```
报批流程：
1. 准备报批材料（报告文本、附件、评审意见）
2. 提交报批申请
3. 等待审批
4. 补充材料（如需）
5. 取得批复文件

报批周期：
- 项目建议书：20 个工作日
- 可行性研究报告：20 个工作日
- 项目申请报告：20 个工作日
- 资金申请报告：20-40 个工作日
```

## 常见问题与解决方案

### 问题 1：市场预测不准确
**解决方案**：
- 收集多年历史数据
- 使用多种预测方法（时间序列、回归分析、专家预测）
- 考虑政策影响（产业政策、环保政策）
- 进行敏感性分析

### 问题 2：投资估算偏差大
**解决方案**：
- 使用近期类似项目指标
- 考虑价格上涨因素（价差预备费）
- 充分估算二类费用（土地、拆迁、设计、监理）
- 预留基本预备费（10-15%）

### 问题 3：财务评价结果不真实
**解决方案**：
- 使用合理价格（不含税、含税）
- 正确计算折旧（直线法、加速折旧法）
- 正确计算税金（增值税、所得税）
- 考虑资金时间价值（折现率 8-10%）

## 注意事项

1. **数据要真实**：不能虚构数据、夸大效益
2. **预测要科学**：不能使用不合理假设
3. **方案要可比**：多方案比选，不能只做一个方案
4. **评价要客观**：不能只讲优点、不讲风险
5. **结论要明确**：不能模棱两可、似是而非
