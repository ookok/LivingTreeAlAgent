#### 发布流程
```
本地工具
    ↓
计算文件哈希（SHA-256）
    ↓
上传到 P2P 网络（获取 download_url）
    ↓
创建 ToolListing
    ↓
广播到 P2P 网络
    ↓
状态：PUBLISHED
```

#### 安装流程
```
发现工具（discover_tools）
    ↓
选择工具（listing_id）
    ↓
下载工具代码（从 P2P 网络）
    ↓
验证文件哈希
    ↓
写入到 tools/ 目录
    ↓
安装依赖（pip install）
    ↓
注册到 ToolRegistry
    ↓
更新下载次数
```

## TextCorrectionTool 实现 ✅ (02:00)

### 新增文件：`client/src/business/tools/text_correction_tool.py`（约 450 行）

### 核心类

`TextCorrectionTool(BaseTool)` - 错别字纠正工具

### 功能特性

| 功能 | 说明 |
|------|------|
| **错别字识别** | 同音、形近、语法别错 |
| **上下文感知** | 根据上下文推理正确用词 |
| **批量处理** | 支持单次处理多个文本 |
| **自动纠正** | 可选是否自动应用纠正 |
| **结果缓存** | 避免重复纠正相同文本 |

### 核心方法

| 方法 | 说明 |
|------|------|
| `execute(text/texts, context, auto_correct)` | 执行错别字纠正（主入口） |
| `_correct_text_with_llm()` | 使用 LLM 纠正文本 |
| `_build_correction_prompt()` | 构造 LLM 提示词 |
| `_parse_text_response()` | 解析非 JSON 格式的 LLM 响应 |
| `correct_text()` | 纠正单个文本（便捷方法） |
| `correct_texts()` | 纠正多个文本（便捷方法） |

### LLM 调用方式

使用 `GlobalModelRouter` 调用 LLM：
```python
self._router.call_model_sync(
    capability=ModelCapability.REASONING,  # 使用推理能力
    prompt=prompt,
    temperature=0.3,  # 低温度，提高准确性
)
```

### 提示词设计

```
你是一个专业的文字校对专家。请检查以下文本是否有错别字（包括同音错别字、形近错别字、语法错别字）。

上下文：{context}

待检查文本：{text}

请按以下 JSON 格式输出结果：
{
    "has_error": true,
    "corrections": [
        {
            "original": "原始错误词",
            "corrected": "纠正后的词",
            "reason": "错误原因（同音错别字/形近错别字/语法错别字）",
            "confidence": 0.95
        }
    ],
    "corrected_text": "纠正后的完整文本"
}

注意：
1. 如果没有错别字，has_error 设为 false，corrections 设为空数组
2. 置信度低于 0.7 的纠正建议请谨慎给出
3. 专业术语、人名、地名不要随意纠正
4. 只输出 JSON，不要输出其他内容
```

### 注册到 ToolRegistry

文件末尾包含 `auto_register()` 函数，自动注册到 ToolRegistry：
```python
def auto_register():
    registry = ToolRegistry.get_instance()
    tool = TextCorrectionTool()
    success = registry.register_tool(tool)
```

### 使用示例

```python
from client.src.business.tools.text_correction_tool import TextCorrectionTool

tool = TextCorrectionTool()

# 单个文本纠正
result = tool.correct_text("我想吃平果", context="讨论健康饮食")
# 返回：{"original_text": "我想吃平果", "corrected_text": "我想吃苹果", "corrections": [...]}

# 批量处理
texts = ["我想吃平果", "他学习很认直"]
results = tool.correct_texts(texts, context="学生生活")
```
