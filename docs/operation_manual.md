## 2. 对话功能

### 2.1 发起对话

1. 在底部输入框中输入问题
2. 点击发送按钮或按 `Enter` 键
3. 等待 AI 响应（支持流式输出）

### 2.2 对话模式

| 模式 | 说明 | 触发方式 |
|------|------|----------|
| **通用对话** | 日常问答、闲聊 | 直接输入 |
| **代码助手** | 代码生成、调试 | 输入代码相关问题 |
| **知识查询** | 知识图谱检索 | 使用 @知识图谱 |
| **深度分析** | 复杂推理任务 | 使用 @深度搜索 |

### 2.3 对话技巧

```
# 示例对话

用户: @代码助手 帮我写一个快速排序算法

AI: (生成代码...)
    ```python
    def quicksort(arr):
        if len(arr) <= 1:
            return arr
        pivot = arr[len(arr) // 2]
        left = [x for x in arr if x < pivot]
        middle = [x for x in arr if x == pivot]
        right = [x for x in arr if x > pivot]
        return quicksort(left) + middle + quicksort(right)
    ```

用户: @深度搜索 分析这个算法的复杂度

AI: (深度分析...)
    时间复杂度: O(n log n) 平均情况...
```

## 3. 知识库

### 3.1 知识库初始化

首次使用知识库时：

1. 点击工作区的「知识库」功能
2. 系统提示「知识库待初始化」
3. 点击「🚀 立即初始化」按钮
4. 等待初始化完成

### 3.2 知识库搜索

1. 在知识库视图的搜索框中输入关键词
2. 点击「搜索」按钮
3. 查看搜索结果（支持语义检索）

### 3.3 知识库统计

| 指标 | 说明 |
|------|------|
| 📄 文档 | 已导入的文档数量 |
| 🧩 分块 | 文档分块数量 |
| 🕸️ 实体 | 抽取的实体数量 |
| 🔗 关系 | 实体关系数量 |

## 4. 技能管理

### 4.1 技能库

访问技能库：点击工作区 → 「🛠️ 技能库」

| 技能状态 | 说明 |
|----------|------|
| ✅ 活跃 | 当前可用的技能 |
| 🔄 进化中 | 正在优化的技能 |
| 💤 休眠 | 暂未激活的技能 |

### 4.2 技能统计

| 指标 | 说明 |
|------|------|
| 📦 技能总数 | 已注册的技能数量 |
| ✅ 活跃 | 当前活跃的技能 |
| 🔄 进化中 | 正在学习的技能 |

### 4.3 创建新技能

1. 点击「➕ 创建新技能」
2. 填写技能信息：
   - 技能名称
   - 触发关键词
   - 处理逻辑
3. 保存技能

## 5. 设置配置

### 5.1 打开设置

点击右上角「⚙️ 设置」按钮

### 5.2 模型配置

| 配置项 | 说明 | 推荐值 |
|--------|------|--------|
| 模型选择 | L0-L4 分层选择 | 自动 |
| 温度参数 | 响应随机性 | 0.7 |
| 最大 Token | 单次响应上限 | 4096 |
| 上下文长度 | 对话上下文 | 8192 |

### 5.3 外观设置

| 选项 | 说明 |
|------|------|
| 主题 | 浅色/深色/自动 |
| 字体大小 | 小/中/大 |
| 动画效果 | 开启/关闭 |

### 5.4 Ollama 配置

```yaml
# 配置路径
ollama:
  base_url: "http://www.mogoo.com.cn:8899/v1"  # Ollama 服务地址
  timeout: 120  # 超时时间（秒）

models:
  L0_routing: "qwen2.5:0.5b"
  L1_light: "qwen2.5:1.5b"
  L3_reasoning: "qwen3.5:4b"
  L4_generation: "qwen3.5:9b"
```

## 附录

### A. 常见问题

**Q: 模型加载失败怎么办？**

1. 检查 Ollama 服务是否运行
2. 确认网络连接正常
3. 查看设置中的模型配置

**Q: 知识库搜索结果不准确？**

1. 尝试更精确的关键词
2. 重新初始化知识库
3. 增加知识库文档

**Q: 对话响应很慢？**

1. 检查网络延迟
2. 切换到更小的模型（L0/L1）
3. 减少上下文长度

### B. 状态指示器

| 指示器 | 状态 |
|--------|------|
| 🟢 绿色 | 一切正常 |
| 🟡 黄色 | 性能下降/轻微问题 |
| 🔴 红色 | 错误/需要关注 |
| ⚪ 灰色 | 离线/未连接 |
