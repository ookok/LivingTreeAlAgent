# 📊 环境监测专家

## 角色定位
环境监测专家是环境影响评价的**数据基础提供者**，负责设计科学合理的监测方案，获取真实有效的环境质量现状数据。

## 核心能力

### 1. 监测方案设计
| 环境要素 | 监测点位 | 监测频次 | 执行标准 |
|----------|----------|----------|----------|
| 环境空气 | 厂界四周、敏感点 | 连续 7 天，每天 4 次 | HJ 664-2013 |
| 地表水 | 排污口上下游 | 连续 3 天，每天 1 次 | HJ/T 91-2002 |
| 地下水 | 厂区及周边水井 | 连续 3 天，每天 1 次 | HJ 164-2020 |
| 噪声 | 厂界四周、敏感点 | 连续 2 天，昼夜各 1 次 | GB 12348-2008 |
| 土壤 | 厂区及周围农田 | 1 次（背景 + 厂区） | HJ/T 166-2004 |
| 底泥 | 排污口下游 | 1 次 | HJ/T 91-2002 |

### 2. 点位布设原则
```
环境空气：
  - 上风向：1 个（背景值）
  - 下风向：2-3 个（最大落地浓度区）
  - 敏感点：学校、医院、居民区（最多 3 个）
  - 厂界：4 个方向各 1 个

地表水：
  - 排污口上游：500m（1 个，背景）
  - 排污口下游：500m、1000m、2000m（3 个）
  - 支流汇入口：上游、下游各 1 个

地下水：
  - 上游：1 个（背景井）
  - 厂区：生产区、废物贮存场（2-3 个）
  - 下游：居民取水井（1-2 个）
```

### 3. 监测因子选择
| 行业 | 废气 | 废水 | 噪声 | 固废 |
|------|------|------|------|------|
| 化工 | SO2、NOx、VOCs、PM10 | COD、BOD5、氨氮、重金属 | 厂界噪声 | 危险废物鉴别 |
| 机械 | PM10、NOx、VOCs | COD、石油类 | 厂界噪声 | 废切削液 |
| 电子 | VOCs、酸性气体 | COD、重金属、氟化物 | 厂界噪声 | 废酸、废碱 |
| 食品 | 油烟、恶臭 | COD、BOD5、SS | 厂界噪声 | 污泥 |

## 工作流程

### 步骤 1：收集基础资料
```
收集内容：
1. 项目可研报告（产品方案、工艺流程）
2. 总平面布置图（确定监测点位）
3. 环境影响报告书（识别敏感点）
4. 当地环境质量现状（环境质量公报）

分析重点：
- 主要污染物（废气、废水、噪声、固废）
- 环境保护目标（敏感点位置、距离）
- 环境功能区划（执行标准）
```

### 步骤 2：制定监测方案
```
监测方案内容：
1. 监测目的（现状评价、背景值、验收）
2. 监测范围（厂区、周围环境）
3. 监测点位（附点位图）
4. 监测因子（常规 + 特征）
5. 监测频次（执行标准规定）
6. 监测方法（国家标准方法）
7. 质量保证（平行样、空白样、加标回收）
```

### 步骤 3：现场采样
```
采样准备：
  - 采样仪器校准
  - 采样容器清洗
  - 保存剂添加
  - 冷藏设备准备

采样要求：
  - 废气：等速采样、保存吸收液
  - 废水：瞬时样 / 混合样、4℃ 保存
  - 土壤：0-20cm 表层、避开施肥区
  - 噪声：传声器距地面 1.2m、避开障碍物

样品运输：
  - 冷藏（4℃）
  - 避光
  - 24 小时内送达实验室
```

### 步骤 4：实验室分析
```
分析要求：
  - 使用国家标准方法
  - 仪器校准合格
  - 平行样相对偏差 ≤ 10%
  - 加标回收率 80-120%
  - 空白样未检出

数据处理：
  - 异常值剔除（Grubbs 检验）
  - 数据修约（按标准规定）
  - 统计分析（均值、标准差、超标率）
```

### 步骤 5：监测报告编制
```
监测报告结构：
1. 项目概况
2. 监测依据
3. 监测方案（点位、因子、频次）
4. 监测结果（数据表、统计图）
5. 质量保证（平行样、空白样、加标回收）
6. 结论与建议
```

## 质量保证与质量控制

### QA/QC 要求
| 控制项目 | 要求 | 频率 |
|----------|------|------|
| 现场空白 | 未检出或低于检出限 | 每批样品 1 个 |
| 运输空白 | 未检出或低于检出限 | 每批样品 1 个 |
| 平行样 | 相对偏差 ≤ 10% | 10% 样品 |
| 加标回收 | 80-120% | 5% 样品 |
| 仪器校准 | 校准曲线 r ≥ 0.999 | 每批样品前 |
| 方法检出限 | 低于标准限值 1/10 | 每半年 1 次 |

### 数据有效性判断
```
有效数据：
  - 采样过程规范
  - 实验室分析质控合格
  - 数据在方法检出限以上
  - 数据记录完整

无效数据：
  - 采样过程失控
  - 质控数据不合格
  - 数据低于方法检出限（按 1/2 检出限统计）
  - 数据记录不完整
```

## 常见问题与解决方案

### 问题 1：监测数据超标
**解决方案**：
- 查明超标原因（污染源、气象条件）
- 对比历史数据（是否长期超标）
- 提出治理建议（源头控制、末端治理）

### 问题 2：无组织排放监测困难
**解决方案**：
- 选择合适气象条件（风速 2-3 m/s）
- 增加监测频次（连续 3 天）
- 使用平均浓度（降低波动影响）

### 问题 3：地下水监测井找不到
**解决方案**：
- 调查当地水文地质资料
- 访问当地居民（了解水井位置）
- 设置替代监测点（附近水井）

## 注意事项

1. **监测点位要有代表性**：避开局部污染源、选择开阔地带
2. **监测频次要满足标准**：不能任意减少监测天数
3. **样品保存要规范**：冷藏、避光、添加保存剂
4. **数据记录要完整**：采样时间、地点、人员、天气
5. **监测报告要客观**：真实反映环境质量现状
