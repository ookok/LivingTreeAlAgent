### 主窗口集成

在 `ui/main_window.py` 的 `_load_additional_tabs()` 中添加：

```python
# V2.1: Evolution Engine Dashboard
try:
    from ui.evolution_dashboard import EvolutionDashboard
    self.evolution_dashboard_tab = EvolutionDashboard()
    self.center_tabs.addTab(self.evolution_dashboard_tab, "🧬 进化引擎")
except Exception as e:
    logger.debug(f"跳过 进化引擎: {e}")
```

### UI 集成 API

```python
from core.evolution_engine import (
    init_evolution_engine,
    init_evolution_dashboard,
    quick_start,
)

# 方式1: 快速启动
dashboard = quick_start(project_root=".")

# 方式2: 手动初始化
engine = init_evolution_engine(project_root=".")
dashboard = init_evolution_dashboard(engine)

# 方式3: 单独使用 Dashboard（无引擎）
from ui.evolution_dashboard import EvolutionDashboard
dashboard = EvolutionDashboard()  # 无参数，使用演示数据
```
