## 1. 快速开始

### 1.1 系统要求

| 项目 | 要求 |
|-----|------|
| 操作系统 | Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+) |
| 内存 | 建议 8GB+ |
| 存储 | 至少 10GB 可用空间 |
| Python | 3.11+ (已内置) |

### 1.2 安装与启动

**Windows**:
```bash
双击 run.bat 文件即可启动
```

**Linux/Mac**:
```bash
chmod +x run.sh
./run.sh
```

**首次启动**:
- 系统会自动运行配置向导
- 引导您配置 Ollama 地址、默认模型等
- 完成后自动启动应用

### 1.3 启动命令

```bash
# 启动客户端（默认）
python main.py client

# 启动中继服务器
python main.py relay

# 启动 P2P 追踪器
python main.py tracker

# 环境自检
python main.py check

# 检查更新
python main.py update

# 运行配置向导
python main.py config

# 安装模型
python main.py model llama3
```

## 3. 智能体使用

### 3.1 智能体列表

在智能体面板中，您可以看到所有可用的智能体：

| 列 | 说明 |
|-----|------|
| 名称 | 智能体名称 |
| 类型 | 智能体类型 |
| 状态 | 在线/离线 |
| 描述 | 智能体功能描述 |

### 3.2 启动智能体

1. 在智能体列表中选择一个智能体
2. 点击「启动」按钮
3. 智能体状态变为「在线」
4. 可以开始与智能体对话

### 3.3 智能体类型

| 类型 | 说明 | 示例 |
|-----|------|------|
| **通用智能体** | 通用对话助手 | Hermes |
| **专业智能体** | 专业领域助手 | 代码助手、写作助手 |
| **工具智能体** | 工具调用专家 | 文件操作、浏览器自动化 |
| **企业智能体** | 企业服务助手 | 报表生成、数据分析 |

### 3.4 创建智能体

1. 点击「创建智能体」按钮
2. 输入智能体名称
3. 选择智能体类型
4. 配置智能体参数
5. 点击「创建」

## 9. 常见问题

### 9.1 启动问题

**问题**: 应用无法启动
**解决方案**:
1. 运行 `python main.py check` 进行环境自检
2. 检查 Python 版本是否为 3.11+
3. 检查 Ollama 是否安装并运行
4. 查看日志文件 `logs/livingtree.log`

**问题**: 智能体无法启动
**解决方案**:
1. 检查 Ollama 服务是否正常运行
2. 检查网络连接
3. 检查模型是否已下载

### 9.2 性能问题

**问题**: 响应速度慢
**解决方案**:
1. 关闭不需要的智能体
2. 减少同时运行的任务
3. 调整模型参数（增加温度等）
4. 确保系统资源充足

**问题**: 内存占用高
**解决方案**:
1. 清理缓存
2. 限制历史对话数量
3. 关闭后台服务
4. 重启应用

### 9.3 连接问题

**问题**: 无法连接 Ollama
**解决方案**:
1. 检查 Ollama 服务地址配置
2. 确保 Ollama 已启动
3. 检查防火墙设置
4. 尝试重启 Ollama

**问题**: 无法连接服务器
**解决方案**:
1. 检查网络连接
2. 检查服务器地址配置
3. 联系管理员

### 9.4 数据问题

**问题**: 数据丢失
**解决方案**:
1. 检查数据存储路径
2. 恢复备份文件
3. 联系技术支持

**问题**: 配置丢失
**解决方案**:
1. 重新运行配置向导
2. 手动编辑配置文件
3. 恢复默认配置
