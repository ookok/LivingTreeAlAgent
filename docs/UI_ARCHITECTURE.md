# LivingTree AI OS - UI架构设计文档

## 1. 设计理念

### 1.1 核心理念
- **极简主义**：最少界面元素，最大化内容空间
- **对话驱动**：一切交互通过自然语言会话完成
- **上下文感知**：AI理解用户意图，自动执行操作

### 1.2 设计原则
| 原则 | 说明 |
|------|------|
| 少即是多 | 减少输入，减少点击，减少认知负担 |
| 所见即所得 | 聊天即操作，无需切换页面 |
| 模块化 | 功能模块化，可插拔 |
| 自然交互 | 支持语音、手势、意图理解 |

### 3.1 顶部模块栏 (ModuleBar)
```python
class ModuleBar(QWidget):
    """顶部横排模块选择器"""
    
    # 模块列表
    MODULES = [
        ("🌳", "主界面", "home"),
        ("🔍", "深度搜索", "deep_search"),
        ("📚", "知识库", "knowledge_base"),
        ("🎓", "专家训练", "expert_training"),
        ("💻", "智能IDE", "smart_ide"),
        ("✍️", "智能写作", "smart_writing"),
    ]
    
    # 布局：Logo | 模块按钮(横排) | 设置 | 用户
```

### 3.2 左侧会话面板 (SessionPanel)
```python
class SessionPanel(QWidget):
    """左侧会话历史"""
    
    # 组件
    - search_bar: QLineEdit  # 搜索会话
    - session_list: QListWidget  # 会话列表
    - new_chat_btn: QPushButton  # 新建会话
    - market_btn: QPushButton  # 市场入口
    
    # 交互
    - 点击会话 → 加载历史消息
    - 右键会话 → 删除/重命名/导出
```

### 3.3 聊天主区域 (ChatArea)
```python
class ChatArea(QWidget):
    """聊天消息展示区"""
    
    # 组件
    - message_list: QScrollArea  # 消息列表
    - welcome_card: QWidget  # 欢迎卡片（无消息时显示）
    - streaming_bubble: MessageBubble  # 流式消息
    
    # 功能
    - 流式Markdown渲染
    - 代码高亮
    - 图片预览
    - 消息气泡（用户/AI/系统）
```

### 3.4 输入区域 (InputArea)
```python
class InputArea(QWidget):
    """底部输入区域"""
    
    # 组件
    - attach_btn: QPushButton  # 附件按钮（下拉：图片/文档/文件夹）
    - image_btn: QPushButton   # 图片按钮
    - doc_btn: QPushButton     # 文档按钮
    - voice_btn: QPushButton  # 语音按钮
    - input_box: QTextEdit     # 多行输入框
    - send_btn: QPushButton    # 发送按钮
    
    # 快捷键
    - Enter: 发送
    - Shift+Enter: 换行
    - Ctrl+K: 打开附件菜单
```

### 3.5 右侧工作区 (WorkspacePanel)
```python
class WorkspacePanel(QWidget):
    """右侧用户工作区"""
    
    # 组件
    - folder_tree: QTreeWidget  # 文件夹树
    - toolbar: QToolBar  # 工具栏（刷新/新建/上传）
    - breadcrumb: QLabel  # 路径面包屑
    
    # 功能
    - 选择工作空间文件夹
    - 文件浏览/拖拽上传
    - 与聊天交互（如拖拽文件到输入框）
```

## 6. 交互流程

### 6.1 首次启动流程
```
启动 → 欢迎页 → 选择模式
                     │
        ┌────────────┼────────────┐
        ↓            ↓            ↓
    [登录账号]   [注册账号]    [游客体验]
        │            │            │
        └────────────┴────────────┘
                     ↓
              选择工作空间文件夹
                     ↓
              进入主界面
```

### 6.2 模块切换流程
```
点击模块按钮 → 加载模块面板 → 替换右侧工作区
                              ↓
                    模块可通过聊天交互
                              ↓
                    返回主界面 ← 点击Logo
```

### 6.3 消息发送流程
```
输入文本/附件 → 点击发送/按Enter
                        ↓
              创建用户消息气泡
                        ↓
              显示AI思考动画
                        ↓
              流式输出AI响应
                        ↓
              保存到会话历史
```

## 8. 设计规范

### 8.1 间距规范
- 内边距: 16px / 24px
- 组件间距: 8px / 12px / 16px
- 卡片间距: 12px / 16px

### 8.2 圆角规范
- 按钮: 8px
- 卡片: 12px
- 对话框: 16px
- 输入框: 8px

### 8.3 动画规范
- 过渡动画: 200ms ease-in-out
- 悬停效果: 150ms
- 加载动画: 循环旋转 / 脉冲

### 8.4 图标规范
- 使用 Emoji 作为主要图标
- 备用: Material Icons / Font Awesome
- 尺寸: 16px / 20px / 24px
