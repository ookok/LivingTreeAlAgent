#### 分块策略
| 策略 | 适用场景 | 块大小 |
|------|---------|--------|
| `sentence` | 短文本分析 | 1-5句 |
| `paragraph` | 标准文档 | 1-3段 |
| `topic` | 长文档理解 | 主题一致 |
| `semantic` | 深度分析 | 语义完整 |
| `hierarchical` | 多级结构 | 多级嵌套 |

#### 自动选择策略逻辑
```python
def _select_strategy(self, text: str, task: Optional[str]) -> str:
    # 根据任务选择
    if task:
        if '总结' in task or '摘要' in task:
            return "paragraph"
        if '分析' in task or '理解' in task:
            return "semantic"
        if '翻译' in task or '润色' in task:
            return "sentence"
    
    # 根据长度选择
    if len(text) < 500:
        return "sentence"
    elif len(text) < 3000:
        return "paragraph"
    elif len(text) < 10000:
        return "semantic"
    else:
        return "topic"
```

#### 技术要点
1. **压缩 + 分块 组合使用**：
   - 先压缩（减少冗余 70%+）
   - 再分块（保持语义完整）
   - 分块大小可配置（默认 1000 字符）

2. **自动分块触发条件**：
   - `auto_chunk=True`（默认）
   - 压缩后长度 > `chunk_size`（默认 1000 字符）

3. **返回格式**：
   - 如果未分块：`{"compressed_text": "..."}`
   - 如果分块：`{"chunks": [{"index": 0, "content": "...", ...}]}`

4. **块间关系**：
   - 每个块包含 `relations` 字段（关联块索引）
   - 支持块间引用、继承、因果关系追踪

### 技术要点

#### 闭环知识进化流程
```
用户输入长文本/上传文档
    ↓
SemanticChunker 分块
    ↓
VectorAwareGapDetector 发现缺口
    ↓
IntelligentCrawlerScheduler 调度爬取
    ↓
KnowledgeAutopilot 填补缺口
    ↓
向量库更新
    ↓
循环（直到收敛）
```

#### 缺口发现策略
1. **向量相似度**：对文本分块，搜索相似内容，少于3个相似项认为是缺口
2. **聚类分析**：聚类现有知识，找出稀疏区域
3. **图-向量联合**：中心节点但向量稀疏，发现缺失关系
4. **南京本地化**：南京相关查询优先本地来源

#### 爬虫调度策略
| 缺口类型 | 目标来源 |
|-----------|-----------|
| technical_detail | GitHub, StackOverflow, Medium |
| recent_news | Google News, 微信搜一搜 |
| local_info | 南京市政府, 新华报业, 龙虎网 |
| 其他 | Google, 百度 |

def _lazy_decommerce() -> type:
    class DecommercePanel(QWidget):
        def __init__(self, parent=None):
            super().__init__(parent)
            self._load_real_panel()

        def _load_real_panel(self):
            try:
                from client.src.presentation.panels.decommerce_panel import DeCommercePanel
                real_panel = DeCommercePanel(self)
                layout = QVBoxLayout(self)
                layout.setContentsMargins(0, 0, 0, 0)
                layout.addWidget(real_panel)
            except Exception as e:
                print(f"[DecommercePanel] Failed to load: {e}")
                from PyQt6.QtWidgets import QLabel
                layout = QVBoxLayout(self)
                layout.addWidget(QLabel(f"❌ 电商面板加载失败: {e}"))
    
    return DecommercePanel
```

def _lazy_enterprise_cockpit() -> type:
    class EnterpriseCockpitPanel(QWidget):
        def __init__(self, parent=None):
            super().__init__(parent)
            self._load_real_panel()
            
        def _load_real_panel(self):
            try:
                from client.src.presentation.panels.enterprise_cockpit_panel import EnterpriseCockpitPanel as RealPanel
                real_panel = RealPanel(self)
                layout = QVBoxLayout(self)
                layout.setContentsMargins(0, 0, 0, 0)
                layout.addWidget(real_panel)
            except Exception as e:
                print(f"[EnterpriseCockpitPanel] Failed to load: {e}")
                from PyQt6.QtWidgets import QLabel
                layout = QVBoxLayout(self)
                layout.addWidget(QLabel(f"❌ 企业驾驶舱加载失败: {e}"))
    
    return EnterpriseCockpitPanel
```
